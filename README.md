# 🎮 AU TOP DOWN

Protótipo de um jogo 2D Top Down desenvolvido na Unity.

---

## 🕹️ Sobre o jogo

Este é um projeto simples onde o jogador controla um personagem em um ambiente top-down.
O objetivo é sobreviver e explorar, com mecânicas básicas de movimentação e morte ao colidir com inimigos.
